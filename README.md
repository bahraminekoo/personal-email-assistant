# Personal Email Assistant (LangChain + LangGraph)

An AI agent that drafts and responds to emails in your personal style. Beginner-friendly, minimal setup, and ready to extend.

## Features
- Draft replies in your tone using style examples
- Simple LangGraph workflow: read → decide → generate
- CLI to quickly produce replies
- Easy to extend (Gmail integration, memory, templates)

## Quick start
1. Create and activate a virtual environment.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
